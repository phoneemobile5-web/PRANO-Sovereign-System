# **App Name**: AI Workbench

## Core Features:

- AI Project Creation & Management: Users can create, organize, and manage individual AI projects imported from Google AI Studio, maintaining distinct configurations.
- Model & Prompt Configuration Editor: A comprehensive user interface to configure prompts, adjust model parameters, and define input/output schemas for each AI project.
- Interactive AI Playground: An integrated environment to send real-time inputs to configured AI projects and instantly view and interact with their generated responses.
- AI-Powered Prompt Improvement Tool: A generative AI tool that suggests and refines user prompts based on AI best practices and common patterns to optimize model output quality.
- Session History & Bookmarking: Automatic saving of past AI interactions and generated responses, with the ability to bookmark and revisit successful experiments or critical outputs.
- Shareable Project Configurations: Generate unique, read-only links for configured AI projects, allowing users to share their creations for demonstration or collaborative testing.

## Style Guidelines:

- The visual scheme embraces a dark, modern aesthetic, reflective of technology and innovation.
- Primary color: A clear, vibrant blue-purple (`#7070F0`), chosen to convey focus and innovation against the darker backdrop. This color is versatile for interactive elements and key information.
- Background color: A very dark, desaturated blue-purple (`#15151D`), providing a deep and unobtrusive canvas that reduces eye strain, suitable for long coding and design sessions.
- Accent color: A bright, vivid cyan (`#62E0FF`), used to highlight interactive elements, calls to action, and important notifications, creating clear visual contrast.
- Headline and body font: 'Inter' (sans-serif), for its neutral, objective, and highly readable design, fitting for a professional AI development environment.
- Code font: 'Source Code Pro' (monospace), for displaying prompts, code snippets, and model configurations with clarity and excellent legibility.
- Utilize a suite of minimalist, modern, and geometrically inspired line icons to maintain a clean and uncluttered interface, enhancing intuitive navigation.
- Employ a dynamic, multi-panel dashboard layout, allowing users to arrange and resize project lists, editor areas, and output consoles according to their workflow preferences.
- Implement subtle, purposeful animations and state transitions, particularly for AI processing, to provide clear feedback and enhance the perceived responsiveness of the application.
